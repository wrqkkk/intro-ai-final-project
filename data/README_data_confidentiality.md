# Data confidentiality statement

The original dataset used in this project comes from an existing research project and contains restricted individual-level information.

For confidentiality and data-use reasons, the raw data are **not included** in this GitHub repository and will not be submitted as part of the course project.

This repository only contains project documentation, method notes, presentation planning materials, and selected aggregate-level experimental summaries if they are safe to share.

No individual-level raw data, identifiable information, or confidential file paths are included.

The purpose of this repository is to document the AI methods, the modeling logic, the paper reading process, and the final presentation structure.
